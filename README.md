# Obesidad y Enfermedades Metabólicas en América Latina: Prevalencia, Tendencias y Factores de Riesgo por País (2000–2024)

## Descripción

Base de datos sobre la epidemia de obesidad y la carga de enfermedades metabólicas en **países de América Latina** (2000–2024).

**Autor:** Juan Moisés de la Serna Tuya  
**ORCID:** [0000-0002-8401-8018](https://orcid.org/0000-0002-8401-8018)  
**Repositorio OSF:** [https://osf.io/4gu25/](https://osf.io/4gu25/)

## Contenido

1. **Prevalencia de obesidad adulta** (IMC≥30) (%)
2. **Tasas de diabetes tipo 2** (%)
3. **Prevalencia de hipertensión** (%)
4. **Síndrome metabólico** (%)
5. **Niveles de inactividad física** (%)
6. **Tendencias de ingesta calórica** (kcal/día, 2000–2020)

## Países Incluidos

Argentina, Brasil, Chile, Colombia, México, Perú, Bolivia, Ecuador, Venezuela, Uruguay

## Hallazgos Clave

- La obesidad aumentó **+69%** en la región desde 2000
- La diabetes aumentó **+85%** desde 2000
- Se documenta una rápida transición nutricional

## Fuentes de Datos

- **PAHO ENLACE**
